# MEMORY.md 模板

> P0/P1/P2 三层设计。中文关键词之间加空格，帮助 FTS5 正确分词。

```markdown
# MEMORY.md - 长期记忆

## P0 — 每次必读

### 关于 用户
- 时区 Asia/Shanghai
- 偏好：全自动，不喜欢 手动操作
- （在这里记录 用户的 关键信息）

### 服务器
- **主服务器** `x.x.x.x` — 用途，账号/密码
- **备用** `y.y.y.y` — 用途

### 核心规则
- （在这里记录 你和用户 约定的 工作规则）

## P1 — 按需查（指针）

| 内容 | 文件 |
|------|------|
| 凭证/密钥 | `memory/credentials.md` |
| 项目索引 | `memory/PROJECT_INDEX.md` |
| 项目详情 | `memory/projects/项目名.md` |
| 经验教训 | `memory/lessons/` |

## P2 — 配置备忘

- （不常用 但需要 记住的 配置信息）
```

## 设计原则

1. **P0 极度精简**：控制在 2KB 以内，每次启动都会读
2. **中文加空格**：`老婆 刚怀孕` 而非 `老婆刚怀孕`，帮助 FTS5 分词
3. **指针而非内容**：P1 只放文件路径，具体内容在对应文件里
4. **定期清理**：过时的信息及时删除，不让 P0 膨胀
