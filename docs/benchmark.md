## 优化方法汇总

| 优化措施 | 贡献 |
|----------|------|
| 中文关键词空格分词 | FTS5 能正确索引中文 → 零召回问题消除 |
| tags 标签系统 | FTS5 精确命中标签关键词 → 分数提升 |
| vectorWeight 0.75 | 向量搜索主导 → 绕过 FTS5 中文缺陷 |
| chunking 250 tokens | 更小的 chunk → 弱模型更容易匹配 |
| lessons/ 知识库 | 高密度知识文件 → 最高命中率(0.80) |
| 日志压缩 | 减少噪音 → 相关内容更容易浮出 |
